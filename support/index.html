Bus Assistant for Christchurch

Bus Assistant is an app for the Metro bus service in Christchurch that actually works.

It was designed to make using the bus in Christchurch as easy as possible, with great features such as:

  • An easy to use Journey Planner, which helps find the fastest way between A and B
  • A list of nearby stops, to help you quickly identify the nearest stops
  • Real-time and offline bus arrival and departure information for each stop in the network
  • A comprehensive network-wide route map, which also shows all the bus stops in the city
  • Notifications for the Journey Planner, a real-time system which alerts you to actions you need to take to reach your destination
